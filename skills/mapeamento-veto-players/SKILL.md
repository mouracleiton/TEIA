# Mapeamento de Veto Players

## Quando usar
Quando precisar entender QUEM bloqueia uma reforma e COMO superar.

## Estrutura de mapeamento
Para cada veto player, documentar:

1. NOME, partido, UF, cargo
2. MECANISMO DE INFLUÊNCIA: financiamento, porta giratória, base eleitoral, cargo em comissão
3. COMISSÃO: CTF, CAE, CMO, CCJ, CTASP
4. VOTAÇÕES: como votou em pautas relevantes
5. VULNERABILIDADE: jurídica, eleitoral, financeira, ideológica

## Metodologia Tsebelis
Veto player = ator cuja concordância é necessária para mudança do status quo.
- Institucionais: Presidente, Congresso, STF, TCU, BC
- Fáticos: setores que capturam agências, financiam campanhas, judicializam

## Categorias de veto player
1. SETOR FINANCEIRO (bancos, Febraban)
2. ORÇAMENTO SECRETO (relatores, emendas)
3. COMPLEXIDADE (cartórios, contadores, Sistema S)
4. FUNCIONALISMO (sindicatos de servidores)

## Como mapear
- web_search: "doações campanha [nome] TSE 2022"
- web_search: "bancada [setor] congresso nacional"
- web_search: "[nome] votação [tema]"
- TSE DivulgaCandContas para financiamento
- Portal da Câmara/Senado para comissões
