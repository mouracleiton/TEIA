# TEIA Pipeline Dialético

## Quando usar
Quando precisar produzir documento de política econômica que precisa SOBREVIVER a escrutínio técnico hostil.

## Passos
1. FACT-CHECK: cada número verificado contra fonte primária (BCB, STN, IBGE, TSE, CGU)
2. REFUTAÇÃO: documento hostil atacando pontos fracos (Sargent-Wallace, Goodhart, BCB spread)
3. CONTRA-REFUTAÇÃO: para cada ataque, veredito (VENCE/PERDE/EMPATE)
4. SÍNTESE: reescrever incorporando correções

## Armadilhas
- NUNCA somar custos de naturezas diferentes como "poupança redirecionável"
- NUNCA confundir redução de custo de mercado com liberação fiscal
- SEMPRE separar o que precisa de Congresso do que sai via portaria
