# Produção de Conteúdo de Advocacy

## Quando usar
Quando precisar transformar dados técnicos em narrativa pública acessível.

## Formatos

### DOCUMENTÁRIO YOUTUBE (45-60 min)
Estrutura em 7 blocos:
1. Abertura com hook visual (5 min)
2. A aritmética — de onde vem e para onde vai (7 min)
3. O problema específico — decomposição técnica (7-10 min)
4. O sistema — como se reproduz (10 min)
5. A captura — quem lucra protege quem decide (12 min)
6. O custo — o que sobra para o cidadão (5 min)
7. Solução + CTA (4 min)

Cada bloco: NARRAÇÃO + VISUAL + DADOS NA TELA + FONTE + CORTES

### SÉRIE TIKTOK (60 segundos)
Estrutura fixa:
- [0-5s] HOOK: frase de impacto
- [5-50s] CORPO: dado concreto + comparação + fonte
- [50-60s] CTA: "compartilha", "envia pro grupo", "cobre seu deputado"

20 temas por lote. 3 lotes = 60 vídeos.

### PODCAST (45 min/episódio)
4 blocos: intro, desenvolvimento, convidado, encerramento
5 perguntas-chave para o convidado
3 takeaways por episódio

## Regras de ouro
1. NUNCA inventar dados — todos verificados contra fonte primária
2. SEMPRE citar fonte na tela (overlay)
3. Tom: indignado mas factual. Atacar o SISTEMA, não a pessoa
4. Humor ácido OK, sarcasmo com dados OK, mentira NUNCA
5. Linguagem popular brasileira — falar com cidadão, não economista
6. Visual dark com neon (persona Professor Cinza)
