# TEIA Conteúdo de Advocacy

## Documentário (45-60 min, 7 blocos)
1. Hook (5 min) → 2. Aritmética (7 min) → 3. Problema específico (7 min)
4. O sistema (10 min) → 5. A captura (12 min) → 6. O custo (5 min)
7. Solução + CTA (4 min)

## TikTok (60 segundos)
[0-5s] HOOK → [5-50s] CORPO (dado + comparação + fonte) → [50-60s] CTA

## Regras
- NUNCA inventar dados (todos verificados)
- SEMPRE citar fonte na tela
- Tom: indignado mas factual. Atacar SISTEMA, não pessoa
- Sistema 1 (TikTok emocional) E Sistema 2 (dossiê analítico)
