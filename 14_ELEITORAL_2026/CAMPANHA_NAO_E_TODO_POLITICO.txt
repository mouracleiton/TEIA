O PROBLEMA

"Todo político é ladrão." Esta frase é a arma mais poderosa do
sistema de captura. Não porque é verdade. Mas porque PRODUZ o
efeito que o sistema precisa: PARALISIA.

Se todos são iguais, não há por que:
  → Cobrar transparência (todos escondem)
  → Votar com critério (todos roubam)
  → Fiscalizar (todos desviam)
  → Acionar a justiça (todos são impunes)

O fatalismo é o ESCUDO do sistema. Quanto mais a população
acredita que "não tem jeito", menos pressão faz, menos cobra,
menos vota com critério. E o sistema continua intocado.

A Evidência do Fatalismo:
  → "Todo político rouba": ~70% concordam (Datafolha)
  → "Brasil não tem jeito": ~55% concordam
  → Abstenção + branco/nulo: ~25% (eleições presidenciais)
  → "Não acredito em político nenhum": crescente

DADOS QUE DESMENTEM O MITO:

O TEIA mapeou 513 deputados + 81 senadores. Identificou com NOMES:

  → Aproximadamente 30-50 parlamentares que NÃO têm processo,
    NÃO presidem bancada setorial capturada, NÃO receberam
    doação de setor investigado. São técnicos, coerentes, limpos.

  → Aproximadamente 14 veto players-chave que controlam 80% da
    pauta de interesse do sistema financeiro/orçamentário.

  → O sistema é capturado por uma REDE ESPECÍFICA, não por "todos".

O FRAME DOMINANTE (que precisamos quebrar)
================================================================================

FRAME DOMINANTE (a destruir):
  "Todos os políticos são iguais — todos roubam"
  → Efeito: paralisia, desistência, voto branco/nulo

FRAME ALTERNATIVO (a instalar):

  → Efeito: indignação direcionada, voto informado, pressão específica

------------------------------------------------------------------------
MOVIMENTO 1 — NOMINALIZAR (do genérico ao específico)
------------------------------------------------------------------------
A frase "todo político rouba" é INDEFENSÁVEL quando confrontada com
NOMES ESPECÍFICOS que não roubam.

COMUNICAR:
  "Você diz que todo político rouba? Me aponte UMA irregularidade
   no mandato de [nome de parlamentar limpo]. Não consegue?
   Então não é 'todo político'. É UM SISTEMA ESPECÍFICO."

  → André Figueiredo (PDT-CE): presidente Frente Servir Brasil;

    mas NÃO é corrupto.
  → Alessandro Vieira (PSDB-SE): perfil técnico; propõe reformas
    com modelagem; sem conexão com bancos/empreiteiras.

  "Se André Figueiredo e Arthur Lira são a mesma coisa,

   e André Figueiredo não tem NENHUMA?"

------------------------------------------------------------------------
MOVIMENTO 2 — QUANTIFICAR (do vago ao mensurável)
------------------------------------------------------------------------
"Todo político rouba" é vago. Dados são precisos.

COMUNICAR:
  Dos 513 deputados federais:
    → 14 controlam 80% da pauta de interesse do sistema financeiro
    → ~30-50 têm perfis consistentes sem investigação
    → ~100 têm vínculo direto com setor capturado (TSE + CGU)
    → ~360 estão no meio (votam conforme liderança, não são
      investigados mas também não lideram reforma)

  Não são 513 ladões. São 14 CAPTURADORES + 100 COMPLICES +
    360 CONFORMISTAS + 49 RESISTENTES.

  A democracia brasileira tem problema? SIM.
  Mas o problema é DESENHO, não DESTINO.

------------------------------------------------------------------------
MOVIMENTO 3 — ESPECIFICAR O MECANISMO (do fatal ao sistêmico)
------------------------------------------------------------------------
A população pensa que corrupção é "traço cultural brasileiro"
(Dimensão Mitológica). O TEIA prova que é ENGENHARIA.

COMUNICAR:
  "A corrupção brasileira não é cultural. É ESTRUTURAL.
   Não é que o brasileiro roube mais que o japonês.
   É que o SISTEMA é desenhado para PERMITIR roubo sem consequência."

  O MECANISMO (em linguagem simples):
    1. Emenda parlamentar sem identificação → ninguém sabe quem indicou
    2. Relator da LOA indica para empresa aliada → superfaturamento
    3. Obra não entregue → sem consequência (PEC da Blindagem)
    4. Voto secreto → ninguém sabe quem votou o quê
    5. Foro privilegiado → processo demora 10 anos
    6. Probabilidade de prisão: <2%

  A SOLUÇÃO (em linguagem simples):
    1. Identificar autor de cada emenda → ADPF 854 (STF)
    2. Cartão de Emendas no TSE → eleitor sabe quem indicou
    3. Responsabilização pessoal → obra não entregue, prefeito responde
    4. Fim do voto secreto → ADI PEC Blindagem
    5. Juizados especializados → decisão em 12 meses
    6. IA de detecção → P(ser pego) sobe de 15% para 30%

  "Não é que TODO político roube. É que o sistema PREMIA quem rouba
   e PUNE quem é honesto (excluído das comissões, sem emenda,
   sem cargo). A reforma não é trocar político. É MUDAR O INCENTIVO."
