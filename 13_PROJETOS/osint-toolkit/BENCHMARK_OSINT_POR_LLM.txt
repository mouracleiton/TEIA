TEOREMA
Ferramenta OSINT “boa para humano” ≠ “boa para LLM”.
LLM precisa de: CLI/API estável, JSON determinístico, schema documentado,
saída citável, rate-limit previsível, zero GUI obrigatória, licença clara.

4. GAPS CRÍTICOS (por que construir TEIA osint-toolkit)
================================================================================

G1  Stack global OSINT ignora CGU/TSE/BCB/TCU (C6 sistematicamente baixo).
G2  Saídas heterogêneas (HTML, CEF, CSV sem schema) quebram tool-calling.
G3  GUI-first (Maltego) não encaixa em loop agente sem operador.
G4  APIs comerciais exigem chave + budget; agente precisa fallback OSS.
G5  Falta “pacote de citação”: cada fato com fonte URL + timestamp + confiabilidade.
G6  Falta modo SOPBRA-ready: atores → scores de evidência, não só IoCs.

5. ARQUITETURA ALVO (implementada em teia_osint/)
================================================================================

  [LLM / Hermes / outro agente]
            |  tool-call JSON
            v
     teia-osint CLI
       ├─ benchmark   → matriz scores (este doc, machine-readable)
       ├─ collect     → coletores modulares (cgu, bcb, tse, http_json, dns_public)

6. POLÍTICA DE USO (OBRIGATÓRIA)
================================================================================

1. Somente fontes abertas / APIs públicas / dados que o titular publicou.
2. Respeitar robots.txt, ToS e rate limits; backoff exponencial.
3. Não automatizar login, captcha-bypass, scraping de área autenticada.
4. Não coletar dado pessoal sensível além do já público oficial.
5. Toda claim exportada deve carregar URL de evidência.
6. Uso preferencial: accountability, jornalismo, pesquisa, defesa.
7. Proibido: stalking, fraude, acesso indevido, doxxing malicioso.

================================================================================

Camada 0 — Contrato: teia-osint (JSON schema + citações)

Camada 5 — Intel platform (opcional): OpenCTI/Intel Owl se houver ops

Não instalar “VM OSINT completa” dentro do agente.
Preferir coletores finos + JSON + citação.
